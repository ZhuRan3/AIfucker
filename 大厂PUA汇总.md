## 1. 阿里巴巴（阿里味儿）

### 公司别称
- 猫厂、福报厂

### 核心文化特征

#### 阿里"新六脉神剑"价值观
1. 客户第一
2. 团队合作
3. 拥抱变化
4. 诚信正直
5. 激情进取
6. 敬业奉献

#### "阿里血脉"论
- 2024年阿里合伙人樊路远内部演讲引发争议
- 批评灵犀互娱团队"没有流淌着阿里的血脉"
- 被批带有"爹味"，最终道歉并自罚三个月工资

### 典型PUA话术

#### 打压否定型
- **"其实，我对你是有一些失望的。当初给你定级P7，是高于你面试时的水平的。我是希望进来后，你能够拼一把，快速成长起来的。"**
- "你需要有体系化思考的能力"
- "你做的事情，他的价值点在哪里？"
- "你要有owner意识"

#### 期望施压型
- "你真是辜负了我的期待"
- "我对你期望很高，但你现在让我很失望"

#### 精神操控机制
1. **制造心理落差**：先抬高期望值，再表达失望
2. **否定对方能力**：让对方产生自我怀疑
3. **施加隐性压力**：通过"期望"来控制员工行为
4. **制造焦虑感**：让员工认为自己不够优秀

## 2. 腾讯（鹅厂/瑞雪文化）

### 公司别称
- 鹅厂

### 核心文化特征

#### 瑞雪文化
- **起源**：2007年"瑞雪行动"沉淀而来
- **定义**：腾讯人良好行为的代名词
- **覆盖范围**：
  - 道德规范
  - 礼仪规范
  - 职业行为
  - 高压线禁区

#### 具体行为规范
- 不允许逆行乘坐电梯
- 不冲厕所等不文明行为会被说"不瑞雪"
- 设立"瑞雪奖"，颁发"瑞雪大侠"称号

#### 价值观体系
- 愿景、使命
- 价值观（第一条是"正直"）
- 经营理念
- 管理理念

### 腾讯"鹅语"职场术语

| 鹅语 | 含义 |
|------|------|
| 瑞雪 | 文明行为/符合公司价值观 |
| 活水 | 内部转岗制度 |
| 鹅厂 | 腾讯公司别称 |
| 小马哥 | 马化腾 |
| 总办 | 公司最高管理层 |
| BG | 事业群 |
| 同事 | 内部称呼，不叫员工 |

### PUA话术特点
- 相比阿里，腾讯的PUA话术较为隐蔽
- 更多通过"瑞雪"文化来规范员工行为
- 强调"正直"价值观，但实际执行中存在双重标准

### 2018年930变革
- 腾讯第三次重大组织架构变革
- 2019年更新使命愿景，马化腾强调"用户和责任是两条生命线"

## 3. 美团（绩效C文化）

### 公司别称
- 暂无广泛认可别称

### 核心文化特征

#### 绩效管理制度（271/361制度）
- **20%**：A+和A，对应明星员工
- **70%**：B，对应普通员工
- **10%**：C和C-，对应需要绩效改进或淘汰的员工

#### 末位淘汰制度
- 绩效C的员工"有可能一年白干，甚至走人"
- 大厂的末位淘汰是不成文的规定
- 2024年美团严抓考勤，引发员工担忧

### 王兴管理风格

#### "登味"言论
- 2026年，王兴内部讲话：减少"登味"，内部不要再叫他"兴哥"
- 体现扁平化管理风格，但实际执行中仍有层级问题

#### 典型特点
- 强调"对齐"和"复盘"
- 关注"降本增效"
- 管理风格理性、数据驱动

### 美团特色PUA话术

| 话术类型 | 典型语录 |
|----------|----------|
| 绩效威胁 | "绩效C意味着一年白干" |
| 期望施压 | "你要对齐一下颗粒度" |
| 成长绑架 | "这是个很好的成长机会" |
| 考勤施压 | "最近开始严抓考勤了" |

### 争议点
- 严抓考勤引发员工不满
- 绩效C制度带来的焦虑
- "降本增效"背景下的裁员风险

## 4. 字节跳动（字节范儿）

### 公司别称
- 宇宙厂

### 核心文化特征

#### 字节范儿（ByteStyle）
2022年6月更新后的核心价值观：
1. **始终创业**（升至首位）
2. **多元兼容**（提至第二位）
3. **延迟满足**
4. **敢为**与**追求极致**合并
5. **求真**（新增）

#### 企业特点
- 16万员工规模
- 强调平等、UGC形式、个人化和可互动讨论
- 被称为"更受年轻人青睐"
- "入职即巅峰的薪资"

### 张一鸣与互联网黑话
- 张一鸣曾在字节跳动年会上公开讽刺「互联网黑话」现象
- 引发了一轮反黑话运动

### 字节特色PUA话术

| 话术类型 | 典型语录 |
|----------|----------|
| 延迟满足 | "要有延迟满足的能力" |
| 极致追求 | "追求极致，你的颗粒度不够细" |
| 始终创业 | "要保持创业心态" |
| 多元包容 | "要兼容不同观点" |

### 争议事件
- 2024年字节跳动决定不对外投资任何大模型创业公司
- 存在文化冲突和内部创业的矛盾
- AI技术负责人林俊旸离职风波反映大公司成长中的文化分歧

### 常见字节黑话
- 对齐 = 开会
- 闭环 = 自圆其说
- 赋能 = 画饼
- 落地 = 真的干了
- 沉淀 = 没用但留着
- 对焦 = 咱们聊聊
- 拉通 = 拉个群聊
- 复盘 = 批评会

## 5. 华为（狼性文化）

### 公司别称
- 菊厂

### 核心文化特征

#### 狼性文化
任正非解释"狼文化"包含三种精神：
1. **敏感性**：对市场变化敏锐
2. **团队性**：团队协作精神
3. **不屈不挠性**：坚持不懈的奋斗精神

#### 床垫文化
- 员工在办公室放床垫，随时可以加班休息
- 反映华为早期创业时期的艰苦奋斗精神

#### 官方回应
任正非在2020年接受采访时表示：
- 华为**没有996的说法**，更**没有007**
- 华为在劳动合同方面**高于中国标准**
- 最基层员工自愿加班超过一定小时后就不再给予报酬

### 华为特色PUA话术

| 话术类型 | 典型语录 |
|----------|----------|
| 奋斗绑架 | "以奋斗者为本" |
| 狼性施压 | "要有狼性精神" |
| 奉献要求 | "床垫文化是创业精神的体现" |
| 责任绑架 | "公司发展需要你多做贡献" |

### 争议事件
- "胡新宇事件"引发对华为加班文化的广泛讨论
- 有"降薪也要来上班"的说法
- 尽管官方否认996/007，但外界和部分员工反映存在严重加班现象

## 6. 百度（璩静事件）

### 公司别称
- 熊厂、狼厂

### 核心文化特征
- 结果导向型文化
- 强调"狼性"精神
- 技术驱动型公司

### 2024年璩静事件

#### 事件概述
- **时间**：2024年5月
- **人物**：百度副总裁、公关负责人璩静
- **起因**：在抖音发布多条关于职场管理的争议视频
- **结果**：引发轩然大波，璩静道歉并辞职

#### 争议言论（PUA话术典型）
1. **"员工闹分手提离职我秒批"**
2. **"公关员工不可以有假期"**
3. **"我又不是她婆婆，我主要考虑的是她能不能在她的岗位及时给我交付结果"**
4. 表示不需要考虑员工的家庭情况

#### 社会反响
- 被批评为**"冷血"**企业文化
- 被贴上**"狼性文化"**标签
- 引发关于工作与生活平衡的广泛讨论

#### 后续处理
- 李彦宏回应："优秀员工才代表真实的百度"
- HR负责人崔珊珊谈论"捍卫百度文化"

### 百度特色PUA话术

| 话术类型 | 典型语录 |
|----------|----------|
| 结果导向 | "我只看结果，不看过程" |
| 家庭忽视 | "你的家庭情况与我无关" |
| 假期剥夺 | "公关员工没有假期" |
| 秒批离职 | "闹分手提离职我秒批" |

## 9. 职场PUA识别与应对

### 什么是职场PUA

职场PUA（Pick-Up Artist）原意指"搭讪艺术家"，在职场语境下指管理者通过精神操控、情感操纵等手段，让员工失去独立思考能力，成为顺从的"工具人"。

### 职场PUA的四大套路

#### 1. 精神打压
- **目的**：降低员工自信
- **典型话术**：
  - "你太差了，什么都做不好"
  - "你的能力太低了，连最简单的任务都完不成"
  - "我对你有点失望"

#### 2. 情感操纵
- **目的**：让员工成为努力的"工具人"
- **典型话术**：
  - "你真是辜负了我的期待"
  - "我骂你是为了你好"
  - "靠谱的老板一定要跟员工一起同甘共苦"

#### 3. 自我洗脑
- **目的**：让员工失去独立思考能力
- **典型话术**：
  - "你要多磨練"
  - "你要学习成长啊"
  - "公司困难你辛苦点"

#### 4. 压榨劳动力
- **典型制度**：354原则（3人干5人活，拿4人钱）
- **典型话术**：
  - "绝对不会亏待你的"（潜台词：现在正在亏待你）
  - "你是我重点的培养对象"（潜台词：你便宜听话，脏活累活都归你）

### 虚假承诺型PUA

| 话术 | 潜台词 |
|------|--------|
| 绝对不会亏待你的 | 现在正在亏待你，你别闹，没用 |
| 你是我重点的培养对象 | 你便宜听话，脏活累活都归你 |
| 在公司那么多年都有感情了 | 不给你涨薪，不爱待就走 |
| 996是我们修来的福报 | 你应该无条件加班 |

### 996/007工作制PUA话术

- "996是我们这些人修来的福报"（马云）
- "今天中国BAT这些公司能够996，我认为是我们这些人修来的福报"
- "最近工作任务比较重，大家辛苦"

### 如何识别职场PUA

#### 信号1：持续的负面评价
- 你的能力总是被否定
- 无论如何努力都得不到认可
- 经常被拿来与"优秀的同事"比较

#### 信号2：情感操控
- 老板喜怒无常，打一巴掌给个甜枣
- 让你产生愧疚感和感恩心理
- 制造"我是为了你好"的氛围

#### 信号3：信息控制
- 有意打造下属信息壁垒
- 有意拆分下属间信息通道
- 让下属揣度上意

#### 信号4：不合理要求
- 要求无条件加班
- 要求牺牲个人生活
- 要求"感恩戴德"

### 应对策略

#### 心态建设
- **记住**：上司并不是评判你能力和价值的唯一标准
- **明白**：工作是为了生活，不是为了公司牺牲一切
- **坚持**：保持独立思考，不要被洗脑

#### 实用技巧
1. **保留证据**：记录不合理要求的证据
2. **寻求支持**：与同事、朋友、家人交流
3. **设立边界**：明确自己的底线
4. **准备退出**：如果环境恶劣，及时止损

#### 反向PUA话术参考
- "感谢您的建议，我会认真考虑"
- "我理解您的期望，让我们一起探讨可行性"
- "我需要平衡工作和生活，希望得到理解"
