## 项目亮点

### 1. 强度控制系统

用户提出的关键需求：
> "我需要增加一个参数，PUA强度或黑话强度，以便于用户展开不同程度，不同强度的循环，而且是全链路的"

实现的方案：
- 两大维度：黑话强度 × PUA强度 = 25种组合
- 全链路影响：影响语言、Agent行为、Loop机制、审查标准
- 安全边界：防止过度优化、过度删减、矫枉过正

### 2. Agent Team 架构

```
总包 Agent (统筹)
    ├─ 赛马组 (7家分厂 Agent)
    ├─ 分包组 (5类专业 Agent)
    └─ 审查组 (3类审查 Agent)
```

### 3. 内卷循环机制

无论完成与否，永远在复盘、对齐、优化中...
