## 方法论概述

### 🎯 核心目标

本方法论旨在实现以下功能：

1. **风格识别**：识别给定文本属于哪种大厂风格
2. **风格模仿**：用指定大厂风格重写任意文本
3. **风格转换**：将文本从一种风格转换为另一种风格
4. **风格混合**：在对话中体现多种风格的特征

## 风格要素分析框架

### 📐 六维分析模型详解

#### 维度1：词汇密度（Vocabulary Density）

| 维度 | 分析内容 | 评估指标 |
|------|----------|----------|
| **黑话浓度** | 特定术语使用频率 | 高/中/低 |
| **英文掺杂率** | 中英文混用程度 | 0-30% |
| **行业术语** | 专业词汇覆盖面 | 窄/中/宽 |
| **缩写使用** | 缩略语使用习惯 | 多/中/少 |

**评分标准：**
```
黑话浓度 = (黑话词汇数 / 总词数) × 100%
- 高浓度：≥15%
- 中浓度：5-15%
- 低浓度：<5%
```

#### 维度2：句式结构（Sentence Structure）

| 公司 | 典型句式特征 | 示例 |
|------|--------------|------|
| 阿里 | 并列复句、递进结构、多修饰语 | "以A为抓手，赋能B生态，形成C闭环" |
| 腾讯 | 疑问句多、数据导向 | "这个赛道的DAU怎么样？" |
| 字节 | 简洁直接、少修饰 | "在Doc上同步一下" |
| 美团 | 长句、哲学化 | "Think Long Term，长期有耐心" |
| 华为 | 军事化用语、号召式 | "打赢这场战役" |
| 百度 | 技术陈述句 | "这个方案简单可依赖" |
| 网易 | 文艺化、情感化 | "有态度的匠心" |

#### 维度3：语气态度（Tone & Attitude）

```
语气坐标轴：
    紧迫 ────────────────► 从容
    ↑                    ↑
激进  ────────────────►  稳健
```

| 公司 | 语气定位 | 核心态度 |
|------|----------|----------|
| 阿里 | 激进-紧迫 | 赋能、落地、打仗 |
| 腾讯 | 中立-数据 | 看数据、说话 |
| 字节 | 高效-直接 | 简洁、透明 |
| 美团 | 稳健-长远 | 长期、耐心 |
| 华为 | 紧迫-集体 | 奋斗、战役 |
| 百度 | 理性-技术 | 简单、可靠 |
| 网易 | 从容-情感 | 态度、匠心 |

#### 维度4：修辞手法（Rhetorical Devices）

| 手法类型 | 阿里 | 字节 | 华为 | 网易 |
|----------|------|------|------|------|
| 排比 | ✅ | ❌ | ✅ | ❌ |
| 隐喻 | ✅ | ❌ | ✅ | ✅ |
| 重复 | ✅ | ❌ | ✅ | ❌ |
| 对比 | ✅ | ❌ | ✅ | ❌ |
| 设问 | ❌ | ✅ | ❌ | ✅ |
| 引用 | ❌ | ❌ | ✅ | ✅ |

#### 维度5：引用习惯（Citation Habits）

| 公司 | 常引用对象 | 引用频率 | 典型引用 |
|------|-----------|----------|----------|
| 阿里 | 马云语录、内部文档 | 高 | "马老师曾经说过..." |
| 腾讯 | 产品经理、数据 | 中 | "根据数据显示..." |
| 字节 | Doc、OKR | 高 | "在Doc上..." |
| 美团 | 王兴饭否 | 高 | "王兴今天在饭否上说..." |
| 华为 | 任正非讲话 | 高 | "任总说过..." |
| 百度 | 技术文档、论文 | 中 | "根据算法..." |
| 网易 | 丁磊、文艺作品 | 中 | "丁老板说..." |

#### 维度6：格式规范（Format Standards）

| 格式要素 | 阿里 | 字节 | 华为 | 其他 |
|----------|------|------|------|------|
| PPT使用 | 极多 | 少 | 极多(胶片) | 中 |
| 文档长度 | 长 | 短 | 长 | 中 |
| 分点陈述 | 是 | 否 | 是 | 否 |
| 英文首字母大写 | 是 | 是 | 否 | 否 |

### 📋 转换操作步骤

#### 步骤1：原始文本分析

```
输入：待转换文本
分析维度：
1. 检测当前风格（基于六维模型）
2. 提取核心语义（去除风格化表达）
3. 识别关键信息点（事实、数据、要求）
4. 标注风格化元素（待替换的词汇、句式）
```

#### 步骤2：目标风格映射

#### 步骤3：句式重构

```
阿里句式模板：
"以[A]为抓手，赋能[B]生态，形成[C]闭环，沉淀[D]方法论"
→ 例："以用户运营为抓手，赋能平台生态，形成交易闭环，沉淀增长方法论"

字节句式模板：
"这个事情在[Doc]上同步过了，[Double]一下优先级"
→ 例："这个需求在Doc上同步过了，Double一下优先级"

华为句式模板：
"拉通[各方资源]，打赢[这场战役]，实现[目标]"
→ 例："拉通研发一线，打赢这场技术攻坚战，实现自主可控"

"Think Long Term，[长期有耐心]，[做正确的事]"
→ 例："Think Long Term，长期有耐心，做正确的事，其他自然会来"
```
