## 方法论概述

### 核心思想

大厂语言风格由三个维度构成：

```
风格 = 价值观维度 + 黑话密度 + PUA套路强度
```

### 三维评估体系

| 维度 | 取值范围 | 说明 |
|------|----------|------|
| 价值观维度 | 创业/狼性/正直/数据/温和 | 企业文化内核 |
| 黑话密度 | 高/中/低 | 互联网黑话使用频率 |
| PUA强度 | 极强/强/中/弱/无 | 精神操控程度 |

### 风格定位图

```
高黑话    阿里━━━━━━━━━━字节
         ↓
         美团
         ↓
低黑话    腾讯━━━━━━━━━━网易

高PUA    阿里━━━━━━━━━━百度
         ↓
         华为
         ↓
低PUA    字节━━━━━━━━━━腾讯
```

## 语言风格特征矩阵

### 一、阿里巴巴（阿里味儿）

#### 风格DNA
```
价值观: ["客户第一", "团队合作", "拥抱变化", "激情进取"]
核心特征: ["爹味说教", "血脉论", "武侠文化", "体系化思考"]
语调特征: ["居高临下", "恨铁不成钢", "语重心长", "画饼"]
```

#### 句式结构公式

**公式1：失望铺垫式**
```
结构: [其实] + [我对你有一些失望] + [当初XX] + [我是希望XX]
示例: "其实，我对你是有一些失望的。当初给你定级P7，是高于你面试时的水平的。我是希望进来后，你能够拼一把，快速成长起来的。"
```

**公式2：反问质疑式**
```
结构: [你做的事情] + [价值点在哪里] + [你要有XX意识]
示例: "你做的事情，他的价值点在哪里？你要有owner意识。"
```

**公式3：血脉认同式**
```
结构: [流淌着XX的血脉] + [没有XX的基因] + [需要XX化]
示例: "对于灵犀，我认为从一开始就没有流淌着阿里的血脉。"
```

#### 黑话嵌入规则
```
每100字必须包含:
- 至少2个动作类黑话 (赋能/抓手/落地/闭环)
- 至少1个概念类黑话 (底层逻辑/颗粒度/链路)
- 至少1个阿里特色词 (owner/复盘/对齐/沉淀)
```

#### 语气词标记
```
开始标记: 其实/讲真/说实话
过渡标记: 讲到这里/我想强调/你需要理解
结束标记: 好好想想/多复盘/沉淀一下
```

## PUA话术公式库

### 一、精神打压类公式

#### 公式P1：失望铺垫式
```
模板: [其实/说实话]，我对你[有一些/挺]失望的。[当初XX]，[我是希望XX]。

变体A: "其实，我对你是有一些失望的。当初给你定级P7，是高于你面试时的水平的。"
变体B: "说实话，我对你挺失望的。当初招你进来，是看好你的潜力。"
变体C: "讲真，我对你有些失望。当初给你这个机会，是希望你快速成长的。"

关键要素:
1. 起始词: 其实/说实话/讲真
2. 表达失望: 有一些/挺/有些
3. 当初铺垫: 定级/招你进来/给机会
4. 期望表达: 希望XX/看好XX/期待XX
```

#### 公式P2：能力否定式
```
模板: [你/你的XX] + [太差了/不够/缺乏] + [连XX都/无法XX]

变体A: "你的能力太低了，连最简单的任务都完不成。"
变体B: "你太差了，什么都做不好。"
变体C: "你缺乏体系化思考的能力，无法独立负责项目。"

关键要素:
1. 主语: 你/你的能力/你的思考
2. 否定词: 太差/不够/缺乏
3. 对比: 连...都/无法/不能
```

#### 公式P3：价值质疑式
```
模板: [你做的事情] + [价值点在哪里] + [你要有XX意识]

变体A: "你做的事情，价值点在哪里？你要有owner意识。"
变体B: "你这些产出的价值点在哪里？你要有结果导向的意识。"
变体C: "你写这个文档的价值在哪里？你要有用户视角的意识。"

关键要素:
1. 主体: 你做的事情/你的产出/你的XX
2. 质疑: 价值点在哪里/意义是什么
3. 要求意识: 要有XX意识/需要XX思维
```

### 二、情感操纵类公式

#### 公式P4：辜负期待式
```
模板: [你真是/你真的] + [辜负了我的期望/让我很失望]
```

```
关键要素:
1. 施加行为: 骂/批评/严格要求
2. 目的宣称: 为了你好
3. 期望结果: 希望XX/让你XX
```

#### 公式P6：同甘共苦式

```
关键要素:
1. 主体: 靠谱的老板/优秀的团队/我们
2. 行为号召: 一定要/需要/应该
3. 核心词: 同甘共苦/一起奋斗/共渡难关
```

### 四、压榨劳动力类公式

```
关键要素:
1. 承诺词: 绝对不会/一定/肯定
2. 核心词: 不亏待你/给你回报/补偿你
3. 条件: 现在XX/先XX/目前XX
```

#### 公式P11：重点培养式

```
潜台词: 你便宜听话，脏活累活都归你
关键要素:
1. 认定: 你是我
2. 定位: 重点培养对象/看好的员工/核心员工
3. 任务: 这个XX给你/让你负责XX
```

#### 公式P12：感情绑架式
```

变体A: "你在公司那么多年都有感情了，希望留下来。"
```

## 风格转换算法

### 算法概述

```
输入: 原始文本 + 目标风格
输出: 风格转换后的文本

步骤:
1. 分析原始文本的语义结构
2. 提取关键信息点
3. 根据目标风格特征，选择对应的句式模板
4. 嵌入对应密度的黑话
5. 应用对应PUA话术公式（如需要）
6. 调整语气和用词，确保风格一致
```

### 风格转换函数

```python
def convert_style(text, target_style):
    """
    风格转换函数

    Args:
        text: 原始文本
        target_style: 目标风格 (ali/tencent/meituan/bytedance/huawei/baidu/netease)

    Returns:
        转换后的文本
    """

    # 步骤1: 提取语义
    semantics = extract_semantics(text)

    # 步骤2: 选择风格模板
    template = get_style_template(target_style)

    # 步骤3: 嵌入黑话
    jargon_density = get_jargon_density(target_style)
    jargonized = embed_jargon(semantics, jargon_density)

    # 步骤4: 应用PUA公式（如需要）
    if requires_pua(target_style):
        jargonized = apply_pua_formula(jargonized, target_style)

    # 步骤5: 调整语气
    result = adjust_tone(jargonized, target_style)

    return result
```
