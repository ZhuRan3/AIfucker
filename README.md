# Jargon-PUA

> 在某个周五晚上，我们看着满屏的"赋能"、"抓手"、"闭环"，突然产生了一个哲学问题：
> *如果AI也开始用这种语言和我们交流，世界会变成什么样？*

于是，这个项目诞生了。它是一个工具，也是一个玩笑；是一次探索，也是一次致敬。

---

## 🤔 项目背景（为什么要有这种东西？）

### 缘起

在中国互联网大厂的会议室里，每天都在发生着这样的对话：

> "同学，你这个颗粒度不够细啊。要以用户增长为抓手，赋能平台生态，在本地生活赛道形成差异化护城河，最终实现GMV闭环，沉淀可复用的增长方法论。"

——这段话可能在某些人的耳朵里是"专业"的，但在另一些人耳朵里，它是抽象的艺术。

### 我们的想法

我们想做一个**实验**：

1. **研究** - 系统化地分析各厂语言风格，提取规律
2. **复现** - 用AI真实模拟这种说话方式
3. **反思** - 通过模仿来理解和批判
4. **控制** - 用强度控制系统防止"过度优化"

### 我们的承诺

> 本项目纯属学术探讨与艺术创作，如有雷同，纯属巧合。
> 请勿将生成的PUA话术用于实际职场，后果自负。

### 功能清单

| 功能 | 说明 | 典型场景 |
|------|------|----------|
| **风格识别** | 识别文本属于哪个大厂风格 | 分析文档是哪家大厂写的 |
| **风格转换** | 将普通文本转换成指定风格 | 写周报时"阿里化"一下 |
| **PUA生成** | 生成各厂典型的PUA话术 | 研究"老板话术"的语言学特征 |
| **浓度检测** | 检测文本的黑话浓度 | 评估文档的"大厂味"有多浓 |
| **内卷循环** | 持续PUA+优化直到完成 | 体验"无限返工"的感觉 |
| **模式切换** | 一键切换AI交流模式 | 让Claude也学会"阿里味" |

## 🚀 快速开始（怎么用？）

### 安装

```bash
# 克隆项目
git clone https://github.com/your-org/jargon-pua.git
cd jargon-pua

# 安装依赖
pnpm install

# 配置API密钥（支持Claude和OpenAI）
cp .env.example .env
# 编辑.env文件，填入你的API密钥
```

### 基础用法

#### 1. 风格识别

```bash
jargon-pua analyze "我们需要以用户增长为抓手，赋能平台生态"
```

输出：
```
┌──────────────┬──────────┐
│ 公司风格     │ 阿里巴巴 │
├──────────────┼──────────┤
│ 置信度       │ 95.3%    │
└──────────────┴──────────┘
```

#### 2. 风格转换

```bash
jargon-pua convert "这个功能需要优化" --to ali
```

输出：
```
"我们需要找到合适的抓手，赋能这个功能的优化，形成体验闭环，
最终沉淀出可复用的优化方法论。"
```

#### 3. PUA话术生成

```bash
jargon-pua pua criticism --style ali --intensity moderate
```

输出：
```
"其实，我对你是有一些失望的。当初给你定级的时候，
我是希望你能快速成长起来的。你这个颗粒度太粗了，
要有owner意识。"
```

#### 4. 内卷循环（特色功能）

```bash
jargon-pua loop "优化这段代码" --max-iterations 3
```

你会体验到：
- 第1轮：温和建议
- 第2轮：中度批评
- 第3轮：重度PUA
- ...
- "永远可以更优，永远不对齐"

#### 5. 作为Claude Skill使用

```markdown
@jargon-pua 开启黑话模式
@jargon-pua 识别这段话的风格
@jargon-pua 生成一段阿里风格的批评话术
```

## ⚙️ 强度控制系统（防卷机制）

### 为什么需要强度控制？

**真实场景**：
> 小明：帮我优化这段代码
> AI（极端模式）："你写的这是什么垃圾？我对你很失望...重来！"
> 小明：...（删库跑路）

**有了强度控制**：
> 小明：帮我优化这段代码 --intensity mild
> AI（温和模式）："这段代码可能还有优化空间，建议..."
> 小明：好的，我试试（正常工作）

### 强度矩阵

```

           none    light    medium   heavy   extreme

        ├────────┼────────┼────────┼────────┼────────┤
mild    │ 建议   │ 委婉   │ 温和   │ 中等   │ 严厉   │
        │ 建议   │ 风格   │ 建议   │ 建议   │ 风格   │
```

## 🛡️ 安全提示（必读）

### ⚠️ 重要警告

1. **本项目具有反讽性质** - 旨在通过模仿来揭示和反思大厂文化现象
2. **请勿用于实际职场** - PUA话术仅供学习和研究，不得用于操控他人
3. **强度需要控制** - 建议从低强度开始体验，极端模式可能造成不适
4. **注意代码安全** - 重要代码请备份，Loop模式会产生多次修改

### 使用建议

```
✅ 推荐用法：
- 学术研究
- 文化分析
- 文艺创作
- 个人体验

❌ 避免用法：
- 实际职场操控
- 恶意使用
- 伤害他人
- 违法用途
```
